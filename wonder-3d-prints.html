Wonder 3D Prints

Role: Web Designer
Timeline: July 2025
Website: wonder3dprints.com

TL;DR

Designed and developed an ecommerce website for Wonder 3D Prints using Figma for design, Instant for rapid development, and Shopify for the storefront. The project showcased how modern design tools and AI assistance can create efficient ecommerce solutions.

My Role

I worked as the web designer on this project, creating the visual design and user experience for the ecommerce site, ensuring seamless integration between design tools and the Shopify platform.

About the Project

Wonder 3D Prints needed a modern ecommerce website to showcase their unique 3D printed products and enable customers to easily browse and purchase items online.

Figma allowed me to create the vision and the wireframe of the website. Shopify powers the storefront, ensuring secure and scalable ecommerce. Instant allowed me to design the website like a designer, with a UI and workflow similar to Figma and Framer I was able to design a Shopify site efficiently.

I also used ChatGPT, Lovable, and other AI generators for design inspiration, optimizing text for SEO, and analyzing the website from different personas. Using these AI tools allowed me to work faster and have more efficient results.
